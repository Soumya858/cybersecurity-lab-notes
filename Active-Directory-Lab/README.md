# Active Directory Home Lab

## Overview

This project documents the deployment and configuration of a Windows Server 2025 Active Directory environment using VMware Workstation.

## Lab Environment

* VMware Workstation
* Windows Server 2025 Standard Evaluation
* Domain Controller: DC01
* Domain Name: corp.local
* Windows PowerShell
* Active Directory Domain Services (AD DS)

## Objectives

* Deploy Windows Server 2025
* Configure networking
* Install Active Directory Domain Services
* Promote server to Domain Controller
* Create and manage a domain environment
* Gain hands-on experience with enterprise infrastructure components

## Tasks Completed

### Server Deployment

* Installed Windows Server 2025
* Configured server networking
* Renamed server to DC01

### Active Directory Configuration

* Installed Active Directory Domain Services (AD DS)
* Promoted server to Domain Controller
* Created the domain corp.local
* Verified successful domain deployment

### Administration

* Accessed and managed the server through PowerShell
* Configured and validated domain services
* Explored Active Directory administration concepts
