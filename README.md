# Cyber Toolkit Pro

## Description
A professional command-line cybersecurity toolkit built using Python.

## Features
- Multi-threaded port scanning
- Secure password generation
- Hash generation (MD5, SHA256)
- JSON report generation
- CLI support using argparse

## Usage

Scan:
python cyber_toolkit_pro.py --scan scanme.nmap.org

Generate Password:
python cyber_toolkit_pro.py --password 12

Hash Text:
python cyber_toolkit_pro.py --hash hello

## Output
Generates report.json with results.
